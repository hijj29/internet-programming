#Лабораторная работа №12: Развёртывание приложения с использованием Docker Swarm и секретов
## Шаг 1: Создание стека сервисов
Создадим файл docker-compose.yml со следующей конфигурацией:
```yaml
version: '3.9'

services:
  nginx:
    image: nginx:alpine
    deploy:
      replicas: 3
      placement:
        constraints:
          - node.role == manager

  whoami:
    image: containous/whoami
    deploy:
      replicas: 2
      placement:
        preferences:
          - node.role == worker

  redis:
    image: redis:alpine
    deploy:
      resources:
        limits:
          cpus: '0.50'
          memory: 128M
        reservations:
          cpus: '0.25'
          memory: 64M
```
